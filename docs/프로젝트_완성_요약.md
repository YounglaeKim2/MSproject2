## 🚀 즉시 이용 가능

### **서비스 시작**
```bash
# 백엔드 시작
cd SAJU/backend && uvicorn app.main:app --reload --port 8000

# 프론트엔드 시작 (3초 빠른 모드)
cd SAJU/frontend && npm run start:fast
```

### **접속 URL**
- **메인 서비스**: http://localhost:3000
- **API 문서**: http://localhost:8000/docs
