# MSProject2 - 메인 랜딩 페이지

사주팔자와 관상 분석 서비스를 선택할 수 있는 메인 진입점입니다.

## 실행 방법

```bash
cd main-app
npm install
npm start
```

## 접속

- 메인 페이지: http://localhost:4000

## 서비스 연결

- **사주팔자**: http://localhost:3000 (SAJU 디렉토리)
- **관상 분석**: http://localhost:3001 (Physiognomy 디렉토리)

## 기능

- 서비스 선택 인터페이스
- 각 서비스별 기능 소개
- 반응형 디자인
